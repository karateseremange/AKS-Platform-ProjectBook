# STD-000 — Guide de rédaction documentaire

| Propriété | Valeur |
|-----------|--------|
| Document ID | STD-000 |
| Version | 0.2.0 |
| Statut | Review |

# Objet

Définir les règles de rédaction applicables à l'ensemble de la documentation du projet AKS Platform.

# Principes

- La documentation est un livrable.
- La documentation précède l'implémentation.
- La langue officielle de la documentation est le français.
- Le code source et les identifiants techniques sont rédigés en anglais.
- Les documents utilisent Markdown.
- Les liens internes sont relatifs.
- Les documents suivent Semantic Versioning.
