# AKS Platform Project Book

Documentation officielle de l'architecture d'AKS Platform.

## Présentation

Ce dépôt contient l'ensemble de la documentation qui régit la conception, le
développement, l’exploitation et l'évolution d'AKS Platform.

Le **Project Book** constitue la référence officielle du projet. Toute décision
d'architecture, de gouvernance ou de développement y est documentée avant son
implémentation.

Le code source est développé dans un dépôt distinct.

## Structure du dépôt

```text
project-book/   Vision et organisation du projet
adr/            Décisions d’architecture
rfc/            Propositions d’évolution
standards/      Standards applicables
templates/      Modèles documentaires
guides/         Guides opérationnels et utilisateurs
assets/         Ressources documentaires
releases/       Dossiers de validation des versions
```

## Guides opérationnels

L’index des guides est disponible dans [`guides/README.md`](guides/README.md).

Il couvre :

- l’installation et l’administration de la plateforme ;
- l’utilisation quotidienne par le club ;
- l’information des familles concernant le questionnaire mineur.
