# Objet du document

Cette préface introduit AKS Platform et le Project Book.

Le Project Book constitue la spécification d'architecture de référence d'AKS Platform.

# Contexte

Le projet est né de la volonté de remplacer une collection d'outils indépendants par une plateforme cohérente, documentée, maintenable et évolutive.

Un Proof of Concept a permis de valider les choix techniques et méthodologiques avant de repartir d'une feuille blanche.
