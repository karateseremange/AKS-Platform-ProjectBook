# Guides opérationnels AKS Platform

Ces guides décrivent la version **AKS Platform RC 0.7.0** et son module
Questionnaire santé des licenciés mineurs.

## Principes communs

- les réponses détaillées restent confidentielles ;
- elles ne sont jamais communiquées au club ;
- elles ne sont conservées ni dans Google Sheets, ni dans WordPress ;
- seul le représentant légal reçoit leur récapitulatif par e-mail ;
- le club reçoit uniquement l’identité, la référence et la formalité attendue ;
- tout secret, identifiant technique privé ou donnée personnelle doit être exclu de la documentation.

## Versionnement

Les guides suivent les versions validées d’AKS Platform. Toute modification de
configuration ou de procédure doit être revue lors d’une nouvelle RC.
